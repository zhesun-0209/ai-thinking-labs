# Python 代码实验

与网页 **同一案例、同一数字** 的可执行 Python 脚本与 Jupyter Notebook。

## 快速开始（本地）

```bash
cd labs
python3 ch05/search_algorithms.py   # 第 5 章：五种搜索对比表
python3 ch09/bpe.py                 # 第 9 章：BPE 合并步骤
```

可选依赖（第 7 章及以后）：

```bash
pip install -r requirements-min.txt
```

## 目录

```
labs/
  common/           # 与网页共享的数据（JSON）
  ch05/             # 搜索算法
  ch09/             # BPE
notebooks/          # Jupyter Notebook（见 notebooks/README.md）
```

Notebook 从各章正文顶部的 **Python 代码实验** 按钮进入；下载后的 `.ipynb` 内嵌运行所需源码和数据。
