1. MDP 定义¶

状态序列 S0→S1→S2→S3，每步即时奖励 R(s)，折扣 γ 权衡远期回报。

Return

G = R0 + gamma R1 + gamma^2 R2 + ...

2. 价值迭代¶

2.1 Bellman 备份¶

V(s) ← R(s) + γ V(s')，从后向前同步更新。

Bellman

V(s) <- R(s) + gamma * V(s')
